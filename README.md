Cache
=====

Simple caching for JavaEE

#Usage

```xml
    <dependency>
        <groupId>net.froelund</groupId>
        <artifactId>cache</artifactId>
        <version>0.0.1</version>
        <scope>compile</scope>
    </dependency>
```
Inject cache as follows:
```java
    @Inject
    Cache cache;
```
